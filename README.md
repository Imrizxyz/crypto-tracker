# Crypto Tracker 💰

This project fetches real-time cryptocurrency prices using CoinGecko API.

## Features
- Get live price of any cryptocurrency
- Supports multiple coins
- Clean and simple output
- Error handling for invalid coins

## Versions

### V1 (Basic)
- Fetch price of a single coin
- Simple API call and output

### V2 (Improved)
- Supports multiple coins input
- Loop-based processing
- Better output formatting

## Tech Used
- Python
- Requests Library
- CoinGecko API

## How to Run

1. Install required library:
   pip install requests

2. Run the file:
   python crypto_v2.py

3. Enter coin names:
   Example: bitcoin, ethereum, dogecoin

## Example Coins
- bitcoin
- ethereum
- dogecoin

## Note
Make sure to enter correct coin IDs as per Coin
